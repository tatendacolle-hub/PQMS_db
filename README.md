# Patient Queue Management System (PQMS)

## Overview
A web-based system designed to eliminate favoritism in hospital out-patient services using a First-Come-First-Served logic.

## Features
- **Nurse Dashboard:** Register patients and auto-assign queue numbers.
- **Doctor Dashboard:** Manage consultations and record medical details.
- **Live Queue:** Real-time updates for patients on mobile devices.
- **Role-Based Security:** Secure login for different staff roles.

## Tech Stack
- PHP & MySQL
- Laragon (Local Server)
- CSS (Responsive Design)
